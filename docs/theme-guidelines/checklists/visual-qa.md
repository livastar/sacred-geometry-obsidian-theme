# Visual QA Checklist

Use a clean demo vault or sanitized vault. Disable unrelated snippets unless testing a documented snippet.

## Setup

- [ ] Install the theme from the current branch.
- [ ] Enable dark base color scheme.
- [ ] Confirm Sacred Geometry Systems is selected.
- [ ] Confirm optional snippets are disabled for base-theme QA.
- [ ] Record Obsidian version and operating system.

## Notes And Editor

- [ ] Reading view body text is readable.
- [ ] Live Preview body text is readable.
- [ ] Source mode is readable.
- [ ] Headings `h1` through `h6` have clear hierarchy.
- [ ] Internal links and external links are distinguishable.
- [ ] Bold, italic, highlight, and selection states are readable.
- [ ] Inline code and code blocks are readable.
- [ ] Lists, task lists, and tables are readable.
- [ ] Blockquotes do not overpower body text.

## App UI

- [ ] Ribbon is usable.
- [ ] File explorer is readable.
- [ ] Search results are readable.
- [ ] Tabs show active/inactive state.
- [ ] Status bar is readable.
- [ ] Command palette is readable and keyboard focus is visible.
- [ ] Settings modal is readable.
- [ ] Menus, dropdowns, buttons, toggles, and inputs are readable.

## Canvas

- [ ] Starter Canvas opens.
- [ ] Canvas background does not obscure cards.
- [ ] Cards are readable at normal zoom.
- [ ] Cards remain understandable when zoomed out.
- [ ] Focused node state is visible.
- [ ] Hover state is visible.
- [ ] Edge colors are legible.
- [ ] Edge labels are readable.
- [ ] Optional semantic snippet classes work when enabled.

## Graph

- [ ] Local graph background is readable.
- [ ] Global graph background is readable.
- [ ] Nodes and edges have sufficient contrast.
- [ ] Hover/focus/current states are visible.
- [ ] Graph controls are readable.

## Callouts

- [ ] Default callout is readable.
- [ ] `tip` and `flow` states are readable.
- [ ] `info`, `map`, and `agent` states are readable.
- [ ] `warning` and `decision` states are readable.
- [ ] `quote` and `knowledge` states are readable.

## Reduced Motion

- [ ] Enable reduced motion at the OS level.
- [ ] Decorative pulse/glow animations stop.
- [ ] Focus and live states remain visible without motion.

## Mobile-Sized QA

- [ ] Test a narrow viewport or Obsidian mobile.
- [ ] Text does not overflow core panels.
- [ ] Controls remain usable.
- [ ] Canvas controls remain reachable if available.
- [ ] Command palette remains usable.

## Screenshot Capture

- [ ] Capture `assets/screenshot-catalog.png`.
- [ ] Capture README preview images if needed.
- [ ] Confirm screenshots contain no private or sensitive data.
