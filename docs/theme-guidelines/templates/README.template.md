# Sacred Geometry Systems

![Sacred Geometry Systems catalog screenshot](assets/screenshot-catalog.png)

Sacred Geometry Systems is a dark Obsidian theme for living systems maps, Canvas workflows, and complex knowledge ecosystems.

## Who It Is For

Use this theme if your vault contains systems maps, project constellations, agent ecosystems, decision paths, or knowledge clusters that need clear visual hierarchy.

Sacred Geometry Systems is created by **Stanislav Ivanov** as part of the Livastar Observer ecosystem. The theme is designed for people who use Obsidian to map systems, projects, agent workflows, decisions, and knowledge structures with more visual clarity.

## Features

- Dark obsidian palette with ivory text, gold structure, copper boundaries, deep indigo panels, and signal-cyan flow.
- Canvas-first styling for cards, edges, focus states, and semantic map roles.
- Geometry helper classes for rings, fields, spirals, circles, flow cues, and framed panels.
- Graph and callout styling aligned with the same semantic colors.
- Reduced-motion behavior for decorative animation.
- Optional starter Canvas, map-node template, and semantic Canvas snippets.

## Installation

### Obsidian Marketplace

1. Open **Settings > Appearance**.
2. Under **Themes**, choose **Manage**.
3. Search for **Sacred Geometry Systems**.
4. Install and enable the theme.

### Manual

1. Download this repository.
2. Copy the theme folder to:

   ```text
   <vault>/.obsidian/themes/Sacred Geometry Systems/
   ```

3. Restart Obsidian.
4. Open **Settings > Appearance** and select **Sacred Geometry Systems**.

## Canvas Workflow

Optional Canvas assets live in `canvas/`, `templates/`, and `snippets/`. They model missions, core systems, agents, resources, interactions, knowledge clusters, decisions, outcomes, and feedback loops.

## Customization

Stable public tokens live under the `--sg-*` prefix. Start with:

```css
--sg-obsidian
--sg-ivory
--sg-gold
--sg-copper
--sg-deep-indigo
--sg-signal-cyan
```

## Style Settings

Style Settings is planned. Until it ships, customize the theme with CSS snippets that override public `--sg-*` tokens.

## Accessibility

The theme targets high contrast on dark surfaces, visible focus states, and reduced-motion behavior for decorative animation.

## Compatibility

- Theme version: `0.2.0`
- Minimum Obsidian version: `1.5.0`
- Mode: dark only
- Obsidian Publish: not currently advertised
